Чтобы вычислить сколько нужно значений чтобы закодировать сколько то символов применяется формула:
### 2<sup>кол-во бит</sup>=количество значений

или
### N = 2<sup>i</sup>
где N - количество кодируемых значений, i - количество бит, с помощью которого можно столько-то значений.
